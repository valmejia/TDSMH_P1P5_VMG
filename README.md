# TDSMH_P1P5_VMG

## Descripción

Aplicación Android nativa desarrollada en **Java** con **Android Studio** que implementa dos pantallas con las siguientes características:

- **Pantalla 1 (Formulario):** Formulario que contiene un campo para ingresar el nombre y un botón para enviar
- **Pantalla 2 (Saludo):** Recibe el nombre enviado y muestra un mensaje de bienvenida personalizado

## Cómo ejecutar el proyecto
1. Clona el repositorio:
   
   ```bash
   git clone https://github.com/valmejia/TDSMH_P1P5_VMG.git

2. Abre Android Studio.
3. Espera a que se sincronicen las dependencias (Gradle).
4. Conecta un dispositivo físico o inicia un emulador.
5. Haz clic en Run.
